# GAIMSat-1 영상 장치 설계 워크샵

## 📋 과제 개요

**목적**: GAIMSat-1의 영상 서브시스템을 설계하고, 그 내용을 PPT로 정리하여 팀에게 발표

**제출물**: 
- PPT 발표자료 (15~20분 분량)
- 설계 계산서 (Excel 또는 MATLAB)

**평가 기준**:
- 요구사항 추적성 (30%)
- 기술적 타당성 (40%)
- 발표 명확성 (20%)
- 질의응답 (10%)

---

## 🎯 미션 배경 (주어진 조건)

```
위성: GAIMSat-1 (6U CubeSat, 10.2kg)
궤도: 530km 태양동기궤도 (SSO)
미션 기간: 6개월
주 임무: 협력 대상위성(3U) 근접 촬영 및 AI 기반 SSA

운용 시나리오:
  Phase 1: 원거리 탐색 (1km ~ 500m) - 대상 검출
  Phase 2: 중거리 접근 (500m ~ 200m) - 추적 유지
  Phase 3: 근거리 촬영 (200m ~ 150m) - 정밀 이미징

볼륨 제약: 영상장치 전체 1.5U 이하
전력 제약: 피크 5W, 평균 2W 이하
```

**[문제 3-2] 센서 선정**

아래 후보 센서를 비교 분석하시오:

| 센서 | 해상도 | 픽셀 크기 | 셔터 | QE | Read Noise |

2. 구경(Aperture) 결정 (회절한계 vs 픽셀 크기)
3. F-number 최적화
4. 렌즈 구성 제안 (COTS vs 커스텀)

- 태양 조사각: 직각
- 노출 시간: 1ms

**발표 포인트**: 설계 마진과 worst-case 분석
